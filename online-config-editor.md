# 📄在线编辑器

## 我该如何使用在线编辑器？

### 第一步

使用 `/scratchit config editconfig NAME` 命令

### 第二步

打开聊天栏给予你的链接

![](.gitbook/assets/immagine%20%2811%29.png)

### 第三步

编辑你想要的物品

![](.gitbook/assets/immagine%20%288%29.png)

### 第四步

点击 **SAVE**

### **第五步**

在你的游戏 **聊天栏** 内按下 **CTRL+V** ，然后按下 **ENTER**

![](.gitbook/assets/immagine%20%286%29.png)
